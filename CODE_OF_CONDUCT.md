# Contributor Code of Conduct

This project adheres to the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/).

Our community expects all participants to:

1. **Be respectful** — treat all community members with kindness and professionalism.
2. **Be inclusive** — welcome and support people of all backgrounds and identities.
3. **Be constructive** — engage in thoughtful, honest feedback and discussion.
4. **Be collaborative** — work together toward common goals.

Unacceptable behavior includes harassment, discrimination, or conduct that creates an unsafe or uncomfortable environment.

## Reporting

If you witness or experience unacceptable behavior, please report it to the project maintainers. All reports will be taken seriously and handled confidentially.

For more details, see the full [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2_0/code_of_conduct/).
